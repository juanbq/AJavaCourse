Arreglos en Java

Arreglos o vectores, si son de dos dimensiones se llaman arreglos bidimensionales o matrices. Pueden tener n dimensiones.

Se usan llaves cuadradas, parentesis cuadrados o corchetes: [ ].

Uso:
int[] arreglo_Datos;
arreglos_Datos[0]= 12;
arreglo_Datos[1]= 450;
System.out.println(arreglo_Datos[1]);

Los paquetes (o como mantener mi codigo aparte de los demás)

Los paquetes se utilizan para "aislar" de alguna manera nuestro codigo de otras personas.

Pero como identificar mi codigo en un  paquete, sin que utilice el mismo nombre que otra persona? Pues hagamolo como si perteneciera al dominio de intenet:

package com.miempresa.contabilidad;
