public class Test {

public static void main(String args[]){
int x=10;
while(x<20) {
System.out.print("El valor de x es:" + x);
x++;
System.out.print("\n");
}
}
}

do while


public class Test {
public static void main(String args[]){
int x=10;
do {
System.out.print("El valor de x es:" + x);
x++;
System.out.print("\n");
}while(x<20);
}
}




For


Syntaxis:

for (inicializacion; condicion ; actualizacion){
// cuerpo
}

La inicializacion se ejecuta primero, y solo una vez.

Siguiente se evalua la condición, si es verdadero se ejecuta el cuerpo

Posteriormente se ejecuta la actualizacion y

Se vuelve a evaluar la condicion.
